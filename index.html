Ons aanbod

Een ruim aanbod aan cursussen voor kinderen vanaf de basisschoolleeftijd. Maak kennis met de wondere wereld van het programmeren met onderstaande cursussen.

Scratch Scratch is een programmeertaal ontwikkeld door MIT speciaal gericht op jonge kinderen. In deze cursus maken kinderen op een spelende manier kennis met computationeel denken: problemen opdelen in kleine stukjes door ze te vertalen naar instructies voor de computer om zo tot een oplossing te komen.
Python Python kent vele toepassingen en is daarom één van de meest populaire programmeertalen van dit moment. Binnen AI en data science geldt Python bij uitstek als de industriestandaard. Deze cursus is meer geschikt voor de wat oudere leerling.
HTML / CSS In deze cursus kinderen de techniek achter websites. Deze cursus kent een groot creatief en visueel aspect: de opbouw beschrijven aan de hand van HTML en de opmaak aan de hand van CSS. Deze cursus is meer geschikt voor de wat oudere leerling.
